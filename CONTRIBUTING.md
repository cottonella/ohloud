# Contributing to ohloud

Thanks for wanting to make ohloud better! 🐻

## Ground rules

- Run `npm test` and `npm run lint` before opening a pull request.
- Keep changes focused and match the surrounding code style.
- For anything large, open an issue to discuss it first.
